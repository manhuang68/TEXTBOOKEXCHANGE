How to run the Online Shoppinh Portal using PHP and MySQL
1. Download the  zip file

2. Extract the file and copy M08 folder

3.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

4. Open PHPMyAdmin (http://localhost/phpmyadmin)

5. Create a database with name csc324

6. Import M08 file(given inside the zip package in SQL file folder)

7. Open Xampp and click on shell to import the sql file (datablase.sql) which is inside of the sql file

8. Run the script http://localhost/M08
